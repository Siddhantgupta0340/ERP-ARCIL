# ProcureFlow X

Enterprise-style Next.js demo for vendor management, invoice intake, 3-way matching, approvals, payments, ERP sync, and audit trail.

## Run

```bash
npm install
npm run dev
```

## Included

- 100+ mock records across vendors, invoices, matches, approvals, payments, audit logs, and notifications
- 20+ fields per module
- Dark enterprise UI
- App Router + Tailwind setup
